Python dependencies: networkx, matplotlib, pygame

To play: 

```
cd Minotaur-Project
python3 game.py
```

To change maze difficulty, edit line 6 in `game.py`.

```
maze_key = get_maze(size = "small", difficulty = "easy")
```

Possible options are 

`size`
- `small`
- `medium`
- `large`
- `random`

`difficulty`
- `easy`
- `medium`
- `hard`
- `max`
- `random`
